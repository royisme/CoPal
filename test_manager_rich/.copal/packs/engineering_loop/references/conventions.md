# Project Conventions

## Git Strategy

- **Worktree**: Use `copal worktree new <feature>` for isolation.
- **Commit Messages**: Conventional Commits (feat, fix, docs, style, refactor, perf, test, chore).

## Code Quality

- **Python**: Follow PEP 8. Use `black` and `isort`.
- **Testing**: `pytest` for all new features.
