# Research Phase Workflow

## Objective

Validate the plan and gather necessary knowledge/context.

## Steps

1. **Analyze Draft Plan**: specific areas needing validation.
2. **External Research**: Use MCP tools to fetch docs/libraries info.
3. **Internal Research**: Search codebase for similar patterns/usages.
4. **Synthesize**: Create `.copal/artifacts/research.json`.
5. **Update Plan**: Refine `.copal/artifacts/plan.json` based on findings.
6. **Handoff**: Proceed to Confirm phase.
