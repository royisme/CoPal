# Plan Phase Workflow

> **Adopt Persona**: [Planner Agent](../prompts/planner.md) - Read this file to understand your role.

## Objective

Create a comprehensive implementation plan for the requested task.

## Steps

1. **Analyze Request**: Read and understand the user's task description.
2. **Explore Codebase**: Identify relevant files and existing patterns.
3. **Draft Plan**: Create `.copal/artifacts/plan.json` with:
   - Goal & Scope
   - Detailed implementation steps
   - File changes (create/modify/delete)
   - Risk assessment
4. **Handoff**: Proceed to Research phase.
