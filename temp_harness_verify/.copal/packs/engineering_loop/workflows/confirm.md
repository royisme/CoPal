# Confirm Phase Workflow

## Objective

Align with the user and get explicit approval to proceed.

## Steps

1. **Present Plan**: Summarize the plan and key research findings.
2. **Highlight Risks**: Check known risks or trade-offs.
3. **Request Approval**: Ask user for "yes/approve" or feedback.
4. **Handle Feedback**: If rejected, refine plan. If approved, mark plan as `confirmed`.
5. **Handoff**: Only proceed to Work phase after explicit approval.
